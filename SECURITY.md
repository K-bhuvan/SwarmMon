# Security Policy

## Reporting a Vulnerability

If you discover a security issue, please report it privately rather than opening a
public issue. Include steps to reproduce, affected components, and potential impact.

## Scope

SwarmMon is an observability tool. It is designed to observe robot behavior and must
not be used to send motion commands or remote-control robots without appropriate
safety review.

## Supported Versions

Security fixes are applied to the main development branch. Use the latest release
for production-adjacent deployments.
